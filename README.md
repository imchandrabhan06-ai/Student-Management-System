# Student Management System

## 📌 Project Overview
This is a Python console-based Student Management System that stores student records using a JSON file. It allows users to perform CRUD (Create, Read, Update, Delete) operations through a simple menu-driven interface.

## ✨ Features
- Add Student
- View Students
- Search Student
- Update Student
- Delete Student
- JSON File Storage
- Menu-Driven Console Interface

## 🛠 Technologies Used
- Python
- JSON
- File Handling

## ▶️ How to Run

1. Clone the repository

```bash
git clone https://github.com/imchandrabhan06-ai/Student-Management-System.git
```

2. Open the project folder

```bash
cd Student-Management-System
```

3. Run the project

```bash
python main.py
```

## 📷 Output

Console-based menu for managing student records.
